This folder contains the OTU tables as obtained from Dada2 and the functional predictions for each table, organized per dataset. Aditionally, the silva database version 138 is also stored here.

There is one folder per dataset: human, plant, mosquito. Inside each folder, there is:

- An OTU table and it's corresponding fasta file
- A folder called 'Temp' where the functional predictions from Tax4Fun2 (named 'functional_prediction.txt') are stored. To be generated when running the scripts.
